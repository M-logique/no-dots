A simple Telegram bot that removes dots from messages. Made with no concept in mind.
</div>

## Setup

1. Clone this repository.
2. Install dependencies: `npm install`
3. Rename `wrangler.toml.template` to `wrangler.toml` and fill in your Cloudflare and Telegram bot details.
4. Deploy the bot: `npx run deploy`
